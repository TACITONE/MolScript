Software libraries used in MolScript

GLUT

  • http://reality.sgi.com/opengl/glut3/glut3.html contains the distribution for the GLUT library written by Mark Kilgard. Version 3.5 or later is required.
  • See also the book Mark Kilgard has written about OpenGL and the X window system.
  • The GLUT page in the OpenGL organisation web site.

JPEG

  • ftp://ftp.uu.net/graphics/jpeg is the official archive site for the JPEG library from the Independent JPEG Group (IJG). Version 6a or later is required.

PNG

  • ftp://ftp.uu.net/graphics/png is one location where the PNG library may be obtained. Version 0.96 or later is required. The zlib library may also be obtained at this location. Version 1.0.4 of zlib is requried.
  • ftp://ftp.uu.net/pub/archiving/zip/zlib, another location where the zlib library may be obtained. Version 1.0.4 is required.

GIF

  • http://www.boutell.com/gd/ is the original site for the gd 1.3 library (a C API for GIF creation), written by Thomas Boutell. This version of gd does not use LZW compression, so it avoids the legal problems with the Unisys patent on that algorithm.
  • Version 1.2 of gd can be also be used with MolScript without modification, but since this version of gd uses LZW compression, it is not recommended for legal reasons.

Other relevant sites

Raster3D

  • The Raster3D Web Site, with distribution information. Version 2.4b or later is required for use with MolScript v2.1.
  • Ethan A. Merritt, author (with others) and maintainer of Raster3D.

OpenGL

  • The OpenGL Organization web site, with information on implementations, general stuff and the governing OpenGL Architechture Review Board (ARB).
  • The OpenGL WWW Center at SGI.
  • The OpenGL Web Site at SGI.
  • Mesa, by Brian Paul: the OpenGL look-alike 3D-rendering software library.

VRML 2.0

  • The VRML Consortium.
  • The VRML Repository.
  • The VRML97 specification, the ISO codification of VRML 2.0.
  • Cosmo Software, download CosmoPlayer, a VRML viewer.

  • Some applications of VRML in chemistry and biology:
    • Image Library of Biological Macromolecules, IMB Jena.
    • VRML in Chemistry, from Technische Hochschule Darmstadt, Germany.
    • VRML 3D crystal structures by Marcus Hewat at ILL Grenoble, France.
    • Library of 3D Molecular Structures from the NYU/ACF Scientific Visualization Laboratory.

  • The specification of the SGI image file format

GNU software

  • The GNU Project's Web Site.

Protein 3D structure data

  • The Protein Data Bank (PDB) at Brookhaven National Lab, USA. This is the scientific data bank where published protein 3D coordinate data sets are deposited by the scientists and from where the data sets can be retrieved by anyone. The specification of the PDB coordinate file format is available. The 3D data sets can be searched in different ways.
